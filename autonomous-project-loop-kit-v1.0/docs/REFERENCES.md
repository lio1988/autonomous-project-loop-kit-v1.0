# Design references

Original implementation informed by public outer-loop patterns:

- Ralph-style fresh invocations, bounded tasks, and file/Git continuity.
- Continuous coding-agent conductors.
- Repository-local LOOPS.md experiments.
- Official Codex, Claude Code, and Cursor non-interactive CLI documentation.

This kit adds runner-owned selection, path enforcement, control-plane restoration, audit rollback, stable completion fingerprints, and no-push/no-merge defaults.
